# 🛍️ Shopping App

## 📖 Project Overview
ده تطبيق تسوّق معمول بـ Flutter بيخلّي المستخدم يقدر يتصفح منتجات إلكترونية ويشتريها وكمان يشوف عروض وخصومات.

---

## ✨ Features
- تصفح المنتجات الإلكترونية  
- عروض وخصومات خاصة  
- واجهة سهلة الاستخدام  
- أقسام منظمة للمنتجات
